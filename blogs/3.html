I don't know

This week we did some more planning and reviewing. Apparently, we are working in 2 teams now because we still hadn't picked which game our set would be doing yet. Our set broke out into 2 sets to write SWOT analysis for each game. We divide into 3 separate categories: Team, Product, and Process and then each member would take turn and put down what they think the Strength, Weakness, Opportunity, Threat for each category are. We took the team members input and then we tried to come up with solutions to overcome Weakness and Threat sections.
